Task Manager

A Simple Task Manager To Boost Your Productivity

This simple task manager is designed to help you prioritize your tasks and stay on track. With a clean and intuitive interface, you can easily add, edit and delete tasks. You can also mark tasks as completed and view your completed tasks in a separate list. The app is fully responsive, making it accessible on any device. The app is built using HTML, CSS and JavaScript, and is fully customizable to fit the needs of any user.
